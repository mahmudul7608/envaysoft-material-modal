##  Core Flow
1. User clicks **Add Material**
2. Modal opens at the center of the screen
3. User selects a material category
4. User filters by color or searches materials
5. Material list updates dynamically
6. User selects one material (active state shown)
7. User clicks **Insert**
8. Modal closes and selected material is displayed
